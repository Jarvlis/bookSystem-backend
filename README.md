# 图书信息平台 - 后端
## 项目基本信息
项目体验地址：
前端：
- HTML + CSS + JavaScript 三件套
- React 开发框架
- Ant Design Pro 项目模板
- Ant Design 端组件库
- Umi 开发框架
- Umi Request 请求库

- 前端Github地址：https://github.com/Jarvlis/bookSystem-frontend

后端：
- Java8 编程语言
- Spring + SpringMVC + SpringBoot2.7 框架
- MyBatis 数据访问框架
- MySQL 数据库
- jUnit 单元测试库
